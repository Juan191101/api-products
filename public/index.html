Bienvenido a Phone API!

Esta es una API de uso publico para poder consumirla en la creacion y desarrollo de tus aplicaciones, cuando precises datos falsos

Recursos

Actualmente Products API cuenta con los siguientes conjuntos de recursos:

  • /products (27 productos)
  • /users (x usuarios)

Implementacion

Rutas de products

  • /products Ver todos los productos
  • /products/1 Ver un producto con determinado "identificador" (1-27)
  • /products/screen/6.1Ver producto/s con determinado tamaño de pantalla
  • /products/camera/32Ver producto/s con determinados pixeles en la cámara
  • /products/ram/16Ver producto/s con determinada cantidad de memoria ram

Los valores finales son demostrativos, puedes intercambiarlos por otros que estén en existencia, recuerda ser preciso con la ruta del atributo "screen".
